flex 레이아웃

1. display: flex;
    (1) display 속성 값으로 flex를 설정하면 플랙스 컨테이너(플랙스 박스)가 되며, 자식 요소는 플랙스 항목 (flex item)이 된다.
    (2) flex는 하나의 축 ! (행 또는 열)

2. direction : 플랙스 컨테이너의 방향의 주축 방향 -> 박스 방향 !!

6. flex-wrap 속성 : 플랙스 항목을 한 줄 도는 여려 줄로 배치 

7. f;ex-flow 속성
    (1) 플랙스 배치 방향과 여러 줄 배치를 한꺼번에 지정

8. order 속성
    (1) 플랙스 항목의 배치 순서 바꾸기

9. flex 속성

10. justify-content 속성 : 플랙스 항목을 주축 방향(수평)으로 배치할 때의 배치 기준 

11. align-items 속성, align-self 속성 : 교차축을 기준으로 하는 배치 방법 조절
    (1) align-item은 전체 요소에 대한 배치 기준 ?
    (2) align-self는 특정 요소에 대한 배치 ?

12. align-content 속성 : 플랙스 항목이 여러 줄로 표시될 때 교차축 기준의 배치 방법 지정 




grid 레이아웃

1.  grid : display 속성 값으로 grid를 설정하면 그리드 컨테이너가 되며, 행과 열을 가진다. (자식 요소는 grid item이 된다.)
2. grid line : grid의 행/열을 그리는선을 말하며, 각 선은 라인 넘버를 갖는다. grid line은 grid item을 배치하는 기준이 된다.
3. grid track : 그리드 컨데이너의 행 또는 열
4. grid cell : 4개의 grid line이 묶여서 그려지는 가장 작은 단위로 유닛이라 부르기도 한다.
5. grid area : 4개의 grid line이 묶여서 그려지는 공간으로 grid cell들이 묶인 영역
6. grid gutters :  행/열 사이의 간격


grid layout 주요 CSS 속성 : grid layout은 grid container와 grid item에 부여할 수 있는 속성이 구분되어 있다.

연결 선택자 
1. 연결 선택자 : 선잭자와 선택자를 연결해 적용 대상을 한정하는 선택자
2. 컴비네이션 컨텍자 (combination selector) 또는 '조합 선택자' 라고도 함

하위 선택자
1. 부모 요소에 포함된 모든 하위 요소에 스타일이 적용된다.
2. 자식 요소뿐만 아니라 손자 요소, 손자의 손자 요소 등 모든 하위 요소까지 적용
3. 하위 선택자를 정의할 때는 하위 요소와 하위 요소를 나란히 쓴다.

인접 형제 선택자
1. 같은 부모를 가진 형제 요소 중 첫 번째 동생 요소에만 스타일 적용
2. 요소 1과 요소 2는 같은 레벨이면서 요소 1 이후 맨 먼저 오는 요소 2에 스타일을 적용

형제 선택자
1. 형제 요소들에 스타일 적용
2. 인접 형제 선택자와 다른 점은 모든 형제 요소에 다 적용된다는 것 !
