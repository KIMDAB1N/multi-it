    center : 속성 값

주요 선택자

(1) 전체 선택자 (*)
    - 페이지에 있는 모든 요소를 대상으로 스타일을 적용할 때 사용
    - 다른 선택자와 함께 모든 하위 요소에 한꺼번에 스타일을 적용하려고 할 때 주로 사용

(2) 태그 선택자

(3) 클래스 선택자 (문서 안에서 여러번 반복 가능)
    - 문서 안에서 여러 번 반복할 스타일이라면 클래스 선택자로 정의.
    - 마침표(.) 다음에 클래스 이름 지정

(4) id 선택자 (문서 안에서 단 한번만 사용 가능)
    - 문서 한에서 한번만 사용한다면 id 선택자로 정의.
    - 파운드(#) 다음에 id 이름 지정

(5) 그룹 선택자
    - 같은 스타일을 사용하는 선택자를 한꺼번에 정의
    - 쉼표(,)로 구분해 여러 선택자를 나열


캐스케이딩의 원칙

1. 스타일 우선 순위 - 스타일 규칙의 중요도, 적용 범위에 따라 우선순위가 결정되고 우선순위에 따라 위에서 아래로 스타일 적용
    (1) 얼마나 중요한가에 따라
    (2) 얼마나 한정지을 수 있는가에 따라
    (3) 소스 순서에 따라
        - 중요도와 명시도가 같다면 소스 순서에 따라 결정
        - 소스에서 나중에 온 스타일이 먼저 온 스타일을 결정

2. 스타일 상속 - 태그들의 포함 관계에 따라 부모 요소의 스타일을 자식 요소로 위에서 아래로 전달
    (1) 자식 요소에서 별도로 스타일을 지정하지 않으면 부모 요소에 있는 스타일 속성들이 자식 요소로 전달된다.
    (2) 상속을 이용하면 스타일 시트를 효과적으로 만들 수 있다.
    (3) 부모 요소로부터 스타일이 상속되는데 자식 요소에서 다른 스타일을 사용하고자 한다면, 즉, 스타일 충돌이 생긴다면 '중요도'나 '명시도', '소스순서' 등에 따라 우선순위가 결정된다.
    (주의) 스타일의 모든 속성이 부모 요소에서 자식 요소로 상속되는 것은 아니다!
        예시 ) 글자 색은 상속되지만 배경 색은 상속되지 않음


CSS3와 CSS 모듈

01. CSS3와 브라우저 접두사
    (1) CSS3 모듈이 계속 개발되고 있는데 표준 규약이 아닌 속성들은 브라우저에 따라 다른 방식으로 지원된다.
    (2) 속성 이름 앞에 접두사(prefix)를 붙여 브라우저별로 구분.  = vendor prefix
    (3) 표준 규약이 완성된 속성도 옛날 버전의 모던 브라우저 사용자를 고려하기 위해 브라우저 접두사를 붙여 사용하기도 함

    -webkit-        (웹 키트 방식 브라우저용. ex. 사파리, 크롬 등)
    -moz-            (게코 방식 브라우저용 ex. 모질라, 파이어폭스 등)
    -o-                 (opera 브라우저용)
    -ms-              (microsoft 브라우저용 = Internet Explorer)

02. 브라우저 접두사를 자동으로 붙여준다.
-prefix-free.js : 브라우저 벤더 접두사를 자동으로 붙여준다. (구글에서 파일 찾아보셈)


03. 글꼴 관련 스타일 (더 찾아보기)
(1) @font-face 속성

(2) font-size 속성 (더 찾아보기)
    - 글자 크기를 조절하는 속성
    - 사용할 수 있는 값 : 절대 크기, 상대 크기, 숫자, 백분율
    - font-size 속성은 상속된다.

(3) font-weight 속성 
    - 글자 굵기를 조절하는 속성

(4) font-variant 속성 (헐 ㄷㄷ 처음 써봄)
    - 대문자를 소문자 크기에 맞추어 작게 표시 
    - 앗 안예뻐서 잘 안쓰인다고 이런게 있구나~ 정도로만 확인하고 넘어감 어쩐지 ~~

(5) font-style 속성
    - 글자를 이탤릭체로 표현하는 속성

(6) font 속성
    - 스타일을 한 줄에 모두 작성하는 것 
    (형태) font : 블라블라브랄 ~~~ 여러 속성 값 입력;

(7) text-decoration 속성
    - 텍스트에 밑줄을 긋거나 가로지르는 줄 표시
    - 텍스트 링크의 밑줄을 없앨 대도 사용한다.

(8) text-transform 속성
    - 영문 텍스트의 대문자나 소문자를 바꾸는 속성

(9) text-shadow 속성
    - 텍스트에 그림자 효과를 추가하는 속성

(10) white-space 속성

(11) letter-spacing , word-spacing 속성
    - 글자 간의 간격, 단어 간의 간격

(12) direction 속성
    - 텍스트를 쓰는 방향 지정
    - 아랍어처럼 오른쪽에서 왼쪽으로 쓰는 언어일 경우 direction 속성으로 rtl (right to left) 로 지정

(13) text-align 속성
    - 텍스트 정렬 방법 지정
    - text-align은 수평 방향의 정렬을 설정한다.

(14) text-indent 속성
    - 문단의 첫 글자를 얼마나 들여 쓸지 지정

(15) line-height 속성
    - 문단의 줄 간격 지정
    - <숫자>와 <백문율>은 부모 요소를 기준으로 몇 배인지 지정
    - 보통 글자 크기의 1.5~2배 정도면 적당
